LUNAI Project Voice Models Condiciones de uso

1. Al descargar nuestros modelos de voz, aceptas nuestros términos de uso y reglas.
2. El modelo de voz está desarrollado para uso personal y es propiedad intelectual de LUNAI Project y del propietario del personaje. El usuario no puede distribuir una copia a terceros, presentarlo como propio, publicarlo en un sitio web o en cualquier otro medio que permita a otros obtener una copia del modelo de voz.
3. El usuario no debe usar obras creadas con el modelo de voz para fines comerciales sin el permiso de LUNAI Project.
4. El usuario siempre debe indicar claramente y de forma distinta el nombre del modelo de voz utilizado en sus obras.
5. Está prohibido usar el modelo de voz para crear información falsa, voces falsas o desacreditar a otras personas u organizaciones. El usuario debe utilizar el modelo de voz de manera responsable y no violar los derechos de los demás.
6. El usuario no debe usar el modelo de voz para crear, distribuir o almacenar contenido que viole las leyes y normas de la sociedad. Esto incluye insultos, racismo, discriminación, odio, violencia u otras actividades no deseadas. Si tienes alguna duda, contáctanos.
7. LUNAI Project puede proporcionar actualizaciones, correcciones de errores y soporte técnico para el modelo de voz. El usuario puede contactar al equipo de soporte de LUNAI Project para resolver cualquier pregunta o problema.
8. LUNAI Project se reserva el derecho de realizar cambios en los términos de uso y reglas de uso del modelo de voz.

Estos términos de uso están destinados a garantizar un uso seguro y ético del modelo de voz. El usuario debe prestar atención a las reglas y obligaciones para evitar infringir los derechos de los demás y prevenir consecuencias negativas.

¿Contenido para fans?


Eres libre de crear y distribuir ilustraciones, vídeos, carátulas, música y otras obras de arte con los personajes del LUNAI Project, siempre que respeten las normas anteriores.

¿Uso comercial?


Requiere nuestro permiso. Puedes utilizar las canciones que crees con los personajes del LUNAI Project con fines comerciales, siempre que sigan las normas anteriores y tengas nuestro permiso.

¿Contenido R-18?


Sólo se permite contenido ético R-18 para nuestros personajes humanoides. Todos nuestros personajes son canónicamente mayores de 18 años. Si se trata de contenido político, religioso, fetichista, gore u otro contenido sensible, necesitas nuestro permiso.

¿Ilustraciones AI?


Quedan prohibidas las ilustraciones, vídeos o cualquier otra imagen de personajes del LUNAI Project generados por inteligencia artificial.

¿Música AI?


Se prohíben las canciones cuyo acompañamiento musical haya sido creado por inteligencia artificial con la participación de personajes del LUNAI Project.

¿Reutilizar nuestras voces?


Queda prohibida la reutilización de la salida de voz de nuestros modelos en otros motores (porting), así como su redistribución y el entrenamiento de otros modelos de IA.
