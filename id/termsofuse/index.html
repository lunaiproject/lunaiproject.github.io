Ketentuan Penggunaan Model Suara Proyek LUNAI

1. Dengan mengunduh model suara kami, Anda menyetujui persyaratan penggunaan dan aturan kami.
2. Model suara dikembangkan untuk penggunaan pribadi dan merupakan kekayaan intelektual LUNAI Project dan pemilik karakter. Pengguna tidak boleh mendistribusikan salinannya kepada pihak ketiga, menyebarkannya sebagai milik Anda, mempostingnya di situs web atau media lain yang memungkinkan orang lain untuk mendapatkan salinan model suara.
3. Pengguna tidak boleh menggunakan karya yang menggunakan model suara untuk tujuan komersial tanpa izin dari LUNAI Project.
4. Pengguna harus selalu menunjukkan dengan jelas dan jelas nama model suara yang digunakan dalam karya mereka.
5. Dilarang menggunakan model suara untuk membuat informasi palsu, suara palsu, atau mendiskreditkan individu atau organisasi lain. Pengguna harus menggunakan model suara secara bertanggung jawab dan tidak melanggar hak-hak orang lain.
6. Pengguna tidak boleh menggunakan model suara untuk membuat, mendistribusikan, atau menyimpan konten yang melanggar hukum dan norma-norma masyarakat. Hal ini termasuk penghinaan, rasisme, diskriminasi, kebencian, kekerasan, atau aktivitas yang tidak diinginkan lainnya. Jika Anda tidak yakin tentang apa pun, silakan hubungi kami.
7. LUNAI Project dapat menyediakan pembaruan, perbaikan bug, dan dukungan teknis untuk model suara. Pengguna dapat menghubungi tim dukungan LUNAI Project untuk menyelesaikan pertanyaan atau masalah apa pun.
8. LUNAI Project berhak untuk membuat perubahan pada ketentuan penggunaan dan aturan penggunaan model suara.

Ketentuan penggunaan ini dimaksudkan untuk memastikan penggunaan model suara yang aman dan etis. Pengguna harus memperhatikan aturan dan kewajiban untuk mencegah pelanggaran hak orang lain dan konsekuensi negatif.

Konten Penggemar?


Anda bebas membuat dan mendistribusikan ilustrasi, video, sampul, musik, dan karya seni lainnya yang menggunakan karakter LUNAI Project, selama mengikuti aturan di atas.

Penggunaan Komersial?


Membutuhkan izin dari kami. Anda dapat menggunakan lagu yang Anda buat dengan karakter LUNAI Project untuk tujuan komersial, selama lagu tersebut mengikuti peraturan di atas dan Anda mendapatkan izin dari kami.

Konten R-18?


Hanya konten R-18 yang etis untuk karakter humanoid kami yang diperbolehkan. Semua karakter kami berusia di atas 18 tahun. Jika konten tersebut bersifat politis, religius, fetisistis, sadis, atau konten sensitif lainnya, Anda memerlukan izin dari kami.

Ilustrasi AI?


Dilarang menggunakan ilustrasi, video, atau gambar lain dari karakter LUNAI Project yang dihasilkan oleh kecerdasan buatan.

Musik AI?


Dilarang menggunakan lagu yang iringan musiknya dibuat oleh kecerdasan buatan dengan partisipasi karakter LUNAI Project.

Penggunaan sekunder?


Dilarang menggunakan kembali output suara model kami di mesin lain (porting), serta mendistribusikan ulang dan melatih model AI lainnya.
