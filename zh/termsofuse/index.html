LUNAI Project 语音模型使用条款

1. 下载我们的语音模型即表示您同意我们的使用条款和规则。
2. 配音模型为个人使用而开发，属于 LUNAI Project 和角色所有者的知识产权。用户不得向第三方分发拷贝，不得将其冒充为您自己的拷贝，不得将其张贴在网站上或任何其他可能允许他人获得语音模型拷贝的媒介上。
3. 未经 LUNAI Project 许可，不得将使用配音模型的作品用于商业目的。
4. 使用者应始终清楚、明确地标明其作品中使用的声音模型的名称。
5. 禁止使用语音模型制造虚假信息、伪造声音或诽谤其他个人或组织。用户必须负责任地使用语音模型，不得侵犯他人权利。
6. 用户不得使用语音模型创建、传播或存储违反社会法律和规范的内容。这包括侮辱、种族主义、歧视、仇恨、暴力或任何其他不良行为。如果您有任何疑问，请联系我们。
7. LUNAI Project 可能会提供语音模型的更新、错误修复和技术支持。用户可联系 LUNAI Project 的支持团队以解决任何疑问或问题。
8. LUNAI Project 保留更改语音模型的使用条款和规则的权利。

本使用条款旨在确保安全、合乎道德地使用语音模型。用户必须遵守相关规则和义务，防止侵犯他人权利并造成不良后果。

粉丝内容？


在遵守上述规则的前提下，您可以自由创作和发布带有路耐计划角色的插图、视频、封面、音乐和其他艺术作品。

商业用途？


需要我们的许可。只要符合上述规定并获得我们的许可，您就可以将用路耐计划角色创作的歌曲用于商业目的。

R-18 内容？


我们的类人角色只允许有 R-18 级的道德内容。我们的所有角色均已年满 18 周岁。如果涉及政治、宗教、恋物癖、血腥或其他敏感内容，您需要获得我们的许可。

AI 插图？


禁止使用人工智能生成的LUNAI项目角色的插图、视频或任何其他图像。

AI 音乐？


禁止播放由人工智能创作的伴奏歌曲，LUNAI 项目的角色也不得参与。

重复使用我们的声音？


禁止在其他引擎中重复使用我们模型的语音输出（移植），也禁止转发和训练其他人工智能模型。
